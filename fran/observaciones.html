displ: Cilindrada del motor de un coche, en litros. Una variable numérica.

hwy: Eficiencia de combustible de un automóvil en carretera, expresada en millas por galón (mpg). Un automóvil con baja eficiencia de combustible consume más combustible que uno con alta eficiencia al recorrer la misma distancia. Variable numérica.

class: Tipo de coche. Una variable categórica.

ggplot(mpg, aes(x = displ, y = hwy, alpha = class)) +
  geom_point()
#> Warning: Using alpha for a discrete variable is not advised.

Asignar una variable discreta (categórica) no ordenada ( class) a una estética ordenada ( sizeo alpha) generalmente no es una buena idea porque implica una clasificación que en realidad no existe.

# Respuesta gemini: El código no genera puntos azules porque colocaste color = "blue" dentro de la función aes(). En ggplot2, todo lo que se escribe dentro de aes() se interpreta como una variable de los datos que se debe mapear (por lo que R crea una categoría ficticia llamada "blue" y le asigna un color por defecto de su paleta, usualmente rojo); para fijar un color estático para todos los puntos, debes escribir color = "blue" fuera de aes(), es decir, directamente como un argumento de geom_point(aes(x = displ, y = hwy), color = "blue").

9.3 Geometric objects

Formas de linea

# Left
ggplot(mpg, aes(x = displ, y = hwy, shape = drv)) + # forma solo punto, no linea
  geom_smooth()
# Right
ggplot(mpg, aes(x = displ, y = hwy, linetype = drv)) + # así línea!!
  geom_smooth()
# quizás así se ve mejor, 2 geometrias un mismo gráfico:
ggplot(mpg, aes(x = displ, y = hwy, color = drv)) + 
  geom_point() +
  geom_smooth(aes(linetype = drv))

geom_smooth() puedes establecer la groupestética en una variable categórica para dibujar varios objetos.

ggplot(mpg, aes(x = displ, y = hwy)) +
  geom_smooth(aes(group = drv)) #sin leyenda
# Right
ggplot(mpg, aes(x = displ, y = hwy)) +
  geom_smooth(aes(color = drv), show.legend = FALSE)

Puede ser color solo una geometría por ejemplo:

ggplot(mpg, aes(x = displ, y = hwy)) + 
  geom_point(aes(color = class)) + 
  geom_smooth()

Se pueden hacer filtros y graficar diferente con una misma geometría:

ggplot(mpg, aes(x = displ, y = hwy)) + 
  geom_point() + 
  geom_point(
    data = mpg |> filter(class == "2seater"), 
    color = "red"  # para círculo rojo
  ) +
  geom_point(
    data = mpg |> filter(class == "2seater"), 
    shape = "circle open", size = 3, color = "red" # para el borde rojo
  )

Número de filas y número de columnas. ¿Por qué facet_grid() no tiene los argumentos nrow y ncol? Porque facet_grid() define una matriz bidimensional estricta basada explícitamente en las variables que le pasas (una variable define las filas y otra las columnas). Por lo tanto, el número de filas y columnas queda determinado automáticamente por la cantidad de categorías únicas que tienen esas variables, no se puede elegir un número arbitrario.
